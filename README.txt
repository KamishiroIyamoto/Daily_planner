Ежедневник с визуальным интерфейсом.
В репозитории есть:
	1. Папка с dll библиотеками
	2. Папка с иконкой приложения
	3. db файл с базой данных для хранения записей
	4. Два cpp файла с логикой
	5. Два h файла заголовков
	6. ui файл с визуальным интерфейсом на XML
	7. qrc файл для доступа к иконке приложения
	8. txt файл, который вы сейчас читаете
	9. txt файл с лицензией
